Классическая аркадная игра **Arkanoid**, полностью написанная на JavaScript с использованием игрового движка **Phaser 3**. Проект разработан в рамках учебной практики по программированию.

## 📝 Описание

Игра представляет собой динамичный экшн, где игрок управляет платформой (ракеткой) и отражает мяч, чтобы разрушать кирпичи в верхней части экрана. 

Главная особенность реализации — **отсутствие внешних графических и звуковых файлов**. 
* Вся графика (кирпичи, бонусы, платформа, фон) генерируется процедурно через `Phaser.Graphics`.
* Все звуковые эффекты синтезируются в реальном времени с помощью `Web Audio API`.

## ✨ Особенности

-  **Процедурная графика**: Все текстуры рисуются кодом, что минимизирует размер проекта.
-  **Синтезированный звук**: Звуковые эффекты (отскоки, разрушения, бонусы) генерируются математически.
-  **Система бонусов**:
  - **Multi**: Добавляет дополнительный мяч на поле.
  - **Wide**: Временно расширяет платформу.
  - **Speed**: Увеличивает скорость всех активных мячей.
- **Тематическое оформление**: Космическая тематика с процедурным фоном (планета, звезды).
- **Прогрессия сложности**: 5 уникальных уровней с разными цветовыми схемами и конфигурациями блоков.

##  Технологии

- **HTML5 / CSS3**
- **JavaScript (ES6+)**
- **Phaser 3 Framework**
- **Canvas API** (для процедурной отрисовки)
- **Web Audio API** (для синтеза звука)

## 🚀 Как запустить

Проект представляет собой единый HTML-файл.

1. Скачайте файл `game.html`.
2. Откройте файл в любом современном браузере (Firefox, Chrome, Edge).
3. Нажмите **Enter** или кликните мышью для начала игры.

**Примечание:** Проект подгружает библиотеку Phaser 3 через CDN. Для работы требуется подключение к Интернету, либо вы можете скачать библиотеку локально и изменить путь в теге `<script>`.
